LAB_TITLE: Create and Configure an Azure Storage Account
ESTIMATED_DURATION: 30 Minutes
SCREENSHOT_FOLDER: storageacc

In this lab the learner will create a new Azure Storage Account from the Azure Portal inside a pre-provisioned resource group. After the storage account is deployed, the learner will create a blob container inside it, upload a sample text file as a blob, and then change the blob's access tier from Hot to Cool using the portal. The lab ends by verifying the tier change in the blob properties.

TASKS:
  TASK_1: Create a Storage Account
  TASK_2: Create a Blob Container and Upload a File
  TASK_3: Change the Blob Access Tier to Cool and Verify
